# vue-dt-scroll-lock
scroll-lock directive for vue.js(2.0)

## Installation
NPM(recommend)
```
npm install vue-dt-scroll-lock --save
```
## Get started
```
import Vue from 'vue';
import vueScrollLock from 'vue-scroll';

Vue use(vueScrollLock);
```
Directive v-scroll-lock then can be used in any of your Component.
```
<template>
  <div v-scroll-lock>
    <p>test</p>
  </div>
</template>
```
